# Quanto sei Boomer?

Un semplice questionario per testare quanto si è "boomer". Questo è un progetto di test creato per il playground.

## Funzionalità

- Questionario con 3 domande generiche
- Design responsive e pulito
- Alert finale con il risultato

## Tecnologie Utilizzate

- HTML5
- CSS3
- JavaScript vanilla

## Come Utilizzare

1. Apri il file `index.html` nel browser
2. Rispondi alle 3 domande
3. Clicca su "Scopri quanto sei Boomer!"
4. Goditi il risultato!

## Note

Questo è un progetto sperimentale creato per divertimento. Non prendere sul serio i risultati!
